# Task Management Application

A full-stack task management web application for creating, updating, and tracking tasks.

## Features

- User Authentication (Register/Login)
- Create Task
- Update Task
- Delete Task
- View All Tasks
- REST API Architecture
- Responsive Design
- MongoDB Integration

## Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## API Endpoints

### Authentication

POST /api/auth/register

POST /api/auth/login

### Tasks

GET /api/tasks

POST /api/tasks

PUT /api/tasks/:id

DELETE /api/tasks/:id
